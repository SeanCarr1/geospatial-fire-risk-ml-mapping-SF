# Geospatial Machine Learning-Based Fire Risk Assessment for San Francisco

## Project Overview
This project leverages geospatial data science and machine learning to identify, classify, and visualize fire risk zones across San Francisco. By integrating historical fire incident records, supervisor district boundaries, and fire station locations, the project delivers a robust, data-driven framework for urban fire risk assessment and emergency response optimization.

## Key Features
- **Data Integration:** Cleans and merges multi-source datasets (fire incidents, district boundaries, city facilities) using Python, Pandas, and GeoPandas.
- **Feature Engineering:** Extracts risk-related attributes such as incident density and proximity to fire stations.
- **Machine Learning Modeling:** Trains a Random Forest classifier to predict fire risk levels at the incident and district levels.
- **Interactive Visualization:** Visualizes spatial fire risk patterns and emergency resource coverage using Folium, with interactive maps and custom popups.
- **Actionable Insights:** Supports data-driven emergency response, urban planning, and policy-making for safer, more resilient cities.

## Workflow
1. **Data Preprocessing:** Clean, filter, and merge geospatial and tabular datasets.
2. **Exploratory Analysis:** Analyze spatial and temporal patterns in fire incidents.
3. **Feature Engineering:** Calculate incident density, risk levels, and distances to fire stations.
4. **Model Training:** Build and evaluate a machine learning model for fire risk classification.
5. **Visualization:** Create interactive maps to communicate results and support decision-making.

## Technologies Used
- Python (Pandas, GeoPandas, NumPy)
- Machine Learning (scikit-learn)
- Geospatial Analysis (GeoPandas, Folium)
- Data Visualization (Matplotlib, Seaborn, Folium)

## Impact
This project provides actionable, location-specific insights for optimizing fire prevention and emergency response strategies in San Francisco. The methodology is adaptable to other cities and hazards, offering a scalable framework for urban risk assessment and smart city planning.
